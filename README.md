## 🔒 Privacy & Security First

> ✅ **Zero data collection** — Everything runs in your browser only  
> ✅ **No backend / No server** — Pure HTML + CSS + JS  
> ✅ **No cookies** — Nothing stored externally  
> ✅ **No analytics** — Your resume data never leaves your device  
> ✅ **HTTPS enforced** — GitHub Pages provides SSL automatically  
> ✅ **Open source** — Audit the code yourself

## 🚀 How to Use

```
1. Open the live site
2. Select your cybersecurity domain
3. Fill sections one by one (left nav)
4. Watch ATS score update in real-time (right panel)
5. Use Big Tech tab to get company-specific keywords
6. Paste job description for keyword gap analysis
7. Download PDF when score hits 80+ ✅
```
